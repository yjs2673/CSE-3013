0. 해당 프로젝트는 맥북 Xcode 환경을 기준으로 제작. src 폴더의 소스 코드들을 이용해 실행하고 data 폴더의 미로 파일들을 입력받는다.

1. R 버튼을 누르면 미로 파일을 열 수 있도록 파일 창이 뜬다.

2. 미로 파일 하나를 선택해 더블클릭하면 미로가 ofWindow에 출력된다.

3. S 버튼을 누르면 파란색 점이 생기며 게임이 시작된다. 이 게임의 목표는 점을 방향키를 이용해 움직여 미로를 탈출하는 것이다.

4. 이동할 때마다 점수에 직결되는 총 이동 횟수가 증가한다. 단, 이미 이동한 길은 노란색 선으로 출력되며 그 위를 지날 때는 이동 횟수가 증가하지 않는다.

5. 출구를 찾기 힘든 경우, H 버튼을 누르면 도착 지점까지 이어지는 빨간색 선이 출력된다. 이 선은 시작 지점과 최단 경로로 이어져 있다. C 버튼을 누를 경우 힌트 선을 없앨 수 있다.

6. 출구에 도착한 경우, 도착 메세지와 함께 (미로의 크기 - 이동 횟수)가 점수로 표시된다. 이 점수는 해당 플레이 횟수와 함께 상위 15등의 랭크 리스트에 저장, 출력될 수 있다. 점수 계산 공식은 미로의 크기와 상관없이 가중치가 별도로 존재하지 않기에 미로의 크기가 클 수록 높은 점수를 받을 수 있다.

7. 도착 지점에 도달 한 경우 점을 더 이상 이동할 수 없다. S 버튼을 눌러 해당 미로를 재시작하거나 R 버튼을 눌러 새로운 미로 파일을 불러와 게임을 시작할 수 있다.

8. Q 버튼을 누를 경우 미로 게임 프로그램이 종료된다.
